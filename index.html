0玻纖布是什麼

真的是一塊布。 把玻璃拉成直徑 5-9 微米的長纖維（比頭髮細 10 倍），紡成紗，用真正的織布機織成經緯交錯的薄布，外觀半透明、白色泛銀光，厚度 0.03-0.1 mm。剝開一塊主機板的綠色油漆，看到的織紋就是玻纖布本人。

它是 PCB 與 IC 載板的「夾心骨架」，不是貼在晶圓上的東西。 高階 AI 載板有 20-30 層，每一層都是「銅箔 + 環氧樹脂浸潤的玻纖布」三明治。玻纖布像鋼筋混凝土裡的鋼筋 — 看不見、拆不出、決定整塊板的尺寸穩定、機械強度與訊號損耗。

它是結構件，不是耗材。 製程是「織布 → 浸樹脂半烘乾（prepreg）→ 多層疊壓硬化 → CCL → 製成 PCB」。熱壓那一步是單向化學反應，玻纖布永久封死在板裡，跟著伺服器一起進垃圾場。

玻纖紗 → 織布 → 浸樹脂半烘乾 (prepreg) → 多層疊壓 + 銅箔 → CCL 銅箔基板 → 蝕刻成電路 → PCB / IC 載板 → 焊上晶片 → 隨產品報廢
維度說明
物理形態真實的織布，厚度 30-100 微米
原料拉絲玻璃（直徑 5-9 µm 玻纖紗）+ 環氧樹脂
位置IC 載板核心層 + PCB 所有介電層（不貼晶圓）
生命週期隨產品永久存在，不可拆、不可回收
商業意義綁定 AI 伺服器出貨量，非重複購買的耗材
這也是為什麼這次缺料會直接卡住 AI 伺服器出貨數量 — 它不像光阻或 SSD 那種「使用中消耗」的東西，需求量等於「明年要出多少台 AI 伺服器 × 每台幾片載板 × 每片多少層 × 每層多少克玻纖布」。

核心結論

台玻在玻纖布的關鍵地位來自三件事：自研 TS-Glass 低膨脹配方繞過日東紡專利、Low-Dk 第二代電子級玻纖布從紗到布全垂直整合、以及在 NVIDIA / AMD 為首的 AI 供應鏈上是「日系以外唯一可規模化的 dual-source」。

它不是技術絕對領先者，而是世界第二、台廠唯一具備整套高階電子級玻纖布能力的玩家。傳產表皮，AI 上游材料的核心。

1玻纖布在 AI 供應鏈的位置

1.1 物料路徑

玻璃原料 → 玻纖紗 (yarn) → 玻纖布 (cloth) → CCL 銅箔基板 → PCB / IC 載板 → 封裝 (CoWoS / CoWoP) → AI 晶片模組 (GB200, MI300)

玻纖布是 CCL 的骨架，CCL 又是高階 PCB 與 IC 載板的基底。沒有合規格的玻纖布，整條 AI server 板級供應鏈就斷在這一環。

1.2 三個關鍵物理參數

參數名稱為什麼重要AI server 趨勢
Dk介電常數訊號傳輸速度，影響高頻損耗越低越好，用於 PCB 訊號層
Df介電損耗因子訊號傳輸時的能量耗散越低越好
CTE熱膨脹係數多層堆疊時的尺寸穩定性越低越好，用於 IC 載板核心層

AI 加速卡功耗高、訊號頻率高、載板層數多 → 三項同時逼近物理極限，這就是「為什麼一張布難倒全球巨頭」的根本原因。

1.3 GB200 之後的需求結構性放大

NVIDIA GB200 / GB300 的 OAM 與 UBB 板尺寸放大、層數加倍，T-Glass 用量等於原本的雙倍以上，疊加 800G / 1.6T 網通、CoWoP 封裝興起，把過去服務電子業利基市場的玻纖布變成了戰略物資。

2台玻的獨家技術

最關鍵護城河

2.1 TS-Glass — 自研低 CTE 配方

  • 對標日東紡的 T-Glass / NE-Glass（全球低 CTE 玻纖紗約 90% 由日東紡供應）。
  • 台玻自行調整玻璃配方，繞過日東紡相關專利，命名為 TS-Glass。
  • 已通過 NVIDIA 供應鏈三段認證（CCL → 載板廠 → 終端客戶），進入 GB200 量產供應。
  • 商業意義：日東紡新產能要等到 2027 H2 才釋放，這段空窗期 TS-Glass 是市場上唯二可規模供應的低 CTE 替代來源。
這一手是台玻整段故事的支點。 沒有 TS-Glass，台玻就只是「日系廠不夠賣時的次選」；有了 TS-Glass，它變成 NVIDIA 主動找的合作夥伴。

2.2 Low-Dk 第二代電子級玻纖布

  • 第一代：2018 年啟動研發，2020 年隨 5G 設備量產出貨。
  • 第二代：2023 年起針對 AI 應用開發，2025 年完成認證、Q3 進入量產放量。
  • 全球只有 美系（Owens Corning 等）、日系（日東紡、AGC）、台玻 三組玩家掌握第二代規模量產能力。
  • 客戶端：主流 CCL 廠（生益、台光電、聯茂等），終端打進 NVIDIA、AMD，蘋果、微軟亦在搶料名單上。

2.3 Owens Corning 技術授權底層

台玻自 2018 年起與 Owens Corning 進行技術授權合作，引進玻纖漿料配方與製程指導。這層基礎讓台玻在「拉絲穩定性、玻纖紗均勻度、良率」三個工程瓶頸上具備已驗證的工業化能力，是 TS-Glass 與 Low-Dk 二代後續能落地的前提。

2.4 玻纖紗 → 玻纖布的垂直整合

  • 台廠多數玻纖布廠（富喬、建榮、宏和）需要外購紗或部分外購；台玻則是從池窯拉絲、紡紗、織布、後處理一條龍自產。
  • 後段 Low-Dk / Low-CTE 等高階產品對紗的玻璃配方、拉絲條件、纖徑均一度極度敏感 → 能否「自家紗 + 自家布」是良率與性能上限的決定因素。
  • 台玻於台灣與成都皆有產能，建廠週期約 1.5 年，比歐日對手通常需要的 3 年時程快一倍，是這波缺貨循環中最重要的時間優勢。

3.2 台玻獨特定位（一句話）

Positioning
「日系以外規模化的第二來源 + 台廠中唯一全棧（紗→布）+ 唯一通過 NVIDIA 三段認證者」

這個定位在當前缺料循環中價值極高：

  • 客戶 dual-source 需求：國際大廠不會只押日廠，台玻是政策上必選的第二供應商。
  • 建廠速度 1.5 年 vs 對手 3 年：在 2025-2027 缺貨窗口內，台玻是唯一能擴產跟上需求的玩家。
  • 垂直整合：在 Low-Dk 二代 / TS-Glass 這種規格敏感產品上，自家紗讓良率有實質可量化的優勢。

3.3 真實的技術天梯

不要被「台玻打贏日東紡」的標題誤導：

  • 規格絕對前緣：德宏石英纖維紗布 > 日東紡 NE-Glass > 日東紡 T-Glass ≈ 台玻 TS-Glass ≈ Low-Dk 二代 > Low-Dk 一代。
  • 規模 + 客戶廣度：日東紡 >> 台玻 > AGC ≈ Owens Corning > 富喬 > 其他。

台玻的勝利不是技術絕對領先，而是 「規格夠到 GB200 門檻」+「規模能擴」+「日系產能滿」 三件事的交集。

5風險與限制

1. 集團 EPS 拖累：玻纖布雖賺錢，但浮法玻璃本業（中國地產循環）虧損抵銷，財務面表現不會像純玩家漂亮。
2. 石英纖維威脅：德宏的石英纖維紗布在 Dk 上贏 Low-Dk 二代，若高階 PCB 規格再升級，台玻需跟進石英路線。
3. 缺料窗口收斂：日東紡 2027 H2 擴產後，台玻 12 條產線開出時若全球供需反轉，有單一年度過剩風險。
4. 認證綁定客戶：CCL → 載板 → 終端三段認證流程長（約 1.5-2 年），是護城河也是被新對手反追的時間鎖。
5. TS-Glass 專利風險：「繞過日東紡專利」的法律穩固度仍是潛在議題，需追蹤是否有訴訟或交叉授權安排。

6系統觀：三個非顯而易見的觀察

6.1 垂直整合 = 樣本內最佳化

當下游規格越極端（Low-Dk 二代 / TS-Glass），上下游耦合度就越強，垂直整合廠的良率上限自然優於外購紗的競爭者。這是物理層面的 inductive bias，不是組織管理優勢。

6.2 時間鎖即護城河

CCL → 載板 → 終端三段認證是一個結構性慢變數。在 AI 半年一代的快變數市場中，凡是慢變數環節都會被定價成超額利潤，台玻吃的就是這個錯配。

6.3「日系以外」這個位置是政治結構的產物

客戶 dual-source 不是商業偏好而是地緣風險管理；換言之，只要中美與日台局勢持續，台玻的「次優替代」位置本身就有持續性 alpha，這比技術領先更穩。
